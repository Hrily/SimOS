## DINING PHILOSOPHER 

### Input Format

```
N
R
```

N - Number of philosophers

R - Number of Rounds

### Output Format

```
s0 s1 s2 s3 ... sn-1
```

si - State of philosopher, 0 for thinking, 1 for eating

### Sample:


#### Input

```
5
3
```

#### Output

```
0 0 0 1 0
0 0 0 0 1
1 0 0 1 0
```
